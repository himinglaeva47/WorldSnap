# WorldSnap
WorldSnap is a world-space alignment toolkit built for artists working across Maya 2024–2026 and Blender 4.5–5.0.

It was created to solve one of the most frustrating problems in 3D workflows:
assets never come back the way they left.

Whether you're moving meshes between ZBrush, AI mesh generators, or other sculpting tools, scale shifts, pivot changes, and position offsets can break your scene. WorldSnap eliminates that friction by restoring alignment instantly and accurately.

With a single click, you can snap objects together using intelligent world-space logic — matching position, scale, and orientation without manual adjustment.

Key Features:
One-click object snapping (bottom-center, pivot, or custom alignment)
Match size to target using true world-space bounding box scaling
Optional rotation matching for perfect orientation alignment
Ground & center tools for instant scene setup
Advanced pivot tools including polygon-based pivot positioning
Flexible workflow toggles (freeze transforms, delete history, include hierarchy)
Non-destructive options to preserve your scene when needed

Built for Modern Pipelines:

WorldSnap was specifically designed for round-trip workflows, including:

Sculpting tools like ZBrush
AI mesh generators (Tripo, Trellis, and others)
Cross-software asset transfers

No more re-aligning meshes by hand. No more guessing scale.
Bring your assets back exactly where they belong.

🧠 Why It Exists:

Because “close enough” wastes time.

WorldSnap replaces manual adjustment with predictable, repeatable alignment — letting you focus on creating instead of fixing.

WHAT IT DOES

- Snap one object to another
- Match scale using bounding box height
- Align objects to the ground
- Center objects to world origin
- Match rotation between objects
- Support additional objects that follow during alignment
